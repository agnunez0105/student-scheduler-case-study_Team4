# Booked & Busy: Student Scheduler Case Study

This repository contains a UX case study website for Booked & Busy, a scheduling app concept designed to help college students balance classes, study time, clubs, and campus life.

## Team Members

- Arianna Nunez
- Cecilia Bilski
- Alexander Clarke

## Project Goal

The goal of this project is to design a tool that helps students plan earlier, reduce scheduling stress, and balance academic responsibilities with extracurricular involvement.
